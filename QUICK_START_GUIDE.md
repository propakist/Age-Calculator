## 🎯 30-Second Setup

### **Option 1: Run Locally (Easiest)**
1. Download all files to a folder
2. Open `index.html` in your browser
3. ✅ Done! It works instantly

### **Option 2: Deploy on GitHub Pages (Free)**
```bash
1. Go to github.com → Sign up (if needed)
2. Click "New Repository"
3. Name it: propakist/Age-Calculator
4. Upload all files
5. Settings → Pages → Select "main" branch
6. 🎉 Live at: https://yourusername.github.io/Age-Calculator
```

### **Option 3: Deploy on Netlify (Easiest & Fastest)**
```bash
1. Go to netlify.com
2. Click "Deploy manually"
3. Drag & drop all files into the box
4. 🎉 Get live URL in 10 seconds
```

### **Option 4: Deploy on Vercel (Very Easy)**
```bash
1. Go to vercel.com
2. Click "New Project"
3. Upload your files
4. 🎉 Live instantly
```

## 🔧 Customize It (Easy!)

### Change Colors
Edit `styles.css` line 3-8:
```css
--primary: #667eea;        /* Main color */
--secondary: #764ba2;      /* Secondary */
--accent: #f093fb;         /* Accent */
```

### Add More Quotes
Edit `script.js` - add to quotes object:
```javascript
const quotes = {
    she: [
        { text: "Your quote", author: "Author Name" },
        // Add more...
    ],
    // ... etc
};
```

### Change Text
- Edit button text in `index.html`
- Change titles in `index.html`
- Modify error messages in `script.js`

## 🆘 Troubleshooting

**Q: Date picker not showing?**
A: Use Chrome, Firefox, Safari, or Edge (Internet Explorer won't work)

**Q: Calculation seems wrong?**
A: Birth date must be in the past, check your system date

**Q: Styling looks broken?**
A: Clear browser cache (Ctrl+Shift+Del), then refresh

**Q: Not working on mobile?**
A: Make sure files are in same folder, try different browser
